## 種族について

- Hearthian: 現在この星系にいる種族（主人公もHearthian）
- Nomai: はるか昔にこの星系にいた種族


## ゲームの目的（と思われるもの）

主人公が初めて宇宙に出る日、博物館にあったNomaiの像に接触することにより、 \
死亡時にタイムループする体質となった。 \
ループの中でこの星系の謎を解き明かしていく。

- Nomaiの痕跡を探し、彼らに何があったのかを解明する。
- 一定時間が経過すると太陽が超新星爆発し、星系が消滅。（主人公はループ） \
  原因を解明し、これを防ぐ。
- Outer Wilds Ventures（他の星にいる探索メンバー）に会う


## システム・道具

- シグナルスコープ: いろいろな音や信号を拾うことができる。
  - Outer Wilds Ventures: 先輩宇宙飛行士の楽器の音
  - 遭難ビーコン: Nomaiの脱出ポッドの信号
  - 量子ゆらぎ: 巨視的量子現象を起こす物体からの信号

- 巨視的量子現象: \
  物体の位置が定まっておらず、確率的であること。 \
  具体的には、ある物体が視界から外れるとその場所から消え、ランダムに新しい場所に出現するという挙動をする。

- 幽霊物質: \
  ダメージを受ける。肉眼では見えないがリトル・スカウト等で撮影できる。

- リトル・スカウト: \
  調査端末。射出できる。カメラを搭載している。 \
  どこに飛ばしても任意のタイミングで回収可能。

- 翻訳機: \
  Nomaiの文書を翻訳できる。 \
  文書は石碑、テープレコーダー、リング状物体などに遺されている。 \
  青はその場の記録、オレンジは他の場所からきた通信と思われる。

- プロジェクション: \
  プロジェクションストーンをセットすると、 \
  そのストーンの名前に対応する場所の光景が映し出される。

- 重力水晶: \
  水晶の周りで上下方向が変わる。

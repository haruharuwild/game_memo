## 種族について

- Hearthian: 現在この星系にいる種族（主人公もHearthian）
- Nomai: はるか昔にこの星系にいた種族


## ゲームの目的（と思われるもの）

主人公が初めて宇宙に出る日、博物館にあったNomaiの像に接触することにより、 \
死亡時にタイムループする体質となった。 \
ループの中でこの星系の謎を解き明かしていく。

- Nomaiの痕跡を探し、彼らに何があったのかを解明する。
- 一定時間が経過すると太陽が超新星爆発し、星系が消滅。（主人公はループ） \
  原因を解明し、これを防ぐ。
- Outer Wilds Ventures（他の星にいる探索メンバー）に会う


## システム・道具

- シグナルスコープ: いろいろな音や信号を拾うことができる。
  - Outer Wilds Ventures: 先輩宇宙飛行士の楽器の音
  - 遭難ビーコン: Nomaiの脱出ポッドの信号
  - 量子ゆらぎ: 巨視的量子現象を起こす物体からの信号

- 巨視的量子現象: \
  物体の位置が定まっておらず、確率的であること。 \
  具体的には、ある物体が視界から外れるとその場所から消え、ランダムに新しい場所に出現するという挙動をする。

- 幽霊物質: \
  ダメージを受ける。肉眼では見えないがリトル・スカウト等で撮影できる。

- リトル・スカウト: \
  調査端末。射出できる。カメラを搭載している。 \
  どこに飛ばしても任意のタイミングで回収可能。

- 翻訳機: \
  Nomaiの文書を翻訳できる。 \
  文書は石碑、テープレコーダー、リング状物体などに遺されている。 \
  青はその場の記録、オレンジは他の場所からきた通信と思われる。

- プロジェクション: \
  プロジェクションストーンをセットすると、 \
  そのストーンの名前に対応する場所の光景が映し出される。

- 重力水晶: \
  水晶の周りで上下方向が変わる。

- 木の炉辺の衛星（月）。
- Eskerがいる。楽器は口笛。
- Nomaiの遺跡（眼信号探知機）がある。
- 闇のイバラの場所にあった惑星の破片が落ちてきたらしい。


### 砂時計の双子星
- 灰の双子星と燃え盛る双子星という二つの星でできている。
- 灰の双子星から燃え盛る双子星に向かって巨大な砂の柱が立ち上っており、時間経過で砂が移動する。

#### 灰の双子星
- 砂がなくなってくると埋まっていたNomaiの建造物が見えるようになる。　
- 建造物は巨人の大海の探査機射出砲に射出信号を送っていた。灰の双子星プロジェクトと呼ばれていたようだ。

#### 燃え盛る双子星
- Chertがいる。楽器はドラム。
- 量子の月探知機があるらしい。
- 重力砲というものがあるらしい。


### 脆い空洞
- ひび割れた星。
- Riebeckがいる。楽器はバンジョー。
- 空洞のランタンからマグマの隕石が落ちてくる。危険。
- 星の中心がブラックホールになっている。行き先はかなり遠くにあるホワイトホール。
- 量子の月についての手がかりが存在するらしい。
- 南極に眼信号探知機が作られたらしい。

#### 空洞のランタン
- 脆い空洞の衛星。
- マグマに覆われている。


### 巨人の大海
- 海に覆われた星。
- Gabboroがいる。楽器はフルート。
- GabboroもNomaiの像を発見し、ループ体質になった。（あまり協力はしてくれない）
- 地面ごと竜巻に巻き上げられることがある。巻き上げられたら水面に落ちるようにすれば助かる。
- 海上に島が4つあるらしい。
  - Gabboroのいる島
  - 彫像島 \
    Nomaiの彫像がある。岩のアーチでつながった2つの島。
  - 建造ヤードのある島
- 星の中心部は潮流で近づけない。触手の怪物がいるらしい。

#### 軌道探査砲
- 巨人の大海の軌道上を周回している。
- ループ開始時に探査機を射出する。その際に砲が崩壊する。 \
  （ループ開始時の目覚め直後、まさにその様子が見える）
- 探査機の信号を受け取る探査機追跡モジュールという部屋が崩壊の際に分離して行方不明になっている。 \
  プロジェクションによって映ったもの: 黒い枝群、紫の電撃


### 闇のイバラ
- 星の中心からバラのツタを全方向に伸ばしたような形をしている。
- もともとその場所にあった惑星を内側から破壊して現れたらしい。
- Feldsparがいる（かもしれない）。楽器はハーモニカ。


### 侵入者
- 彗星。
- 雪に覆われた部分がある。


### 量子の月
- 巨視的量子現象を起こす星。出現場所がランダム。
- 接触すると通り抜ける。
- 今までの発見場所
  - 巨人の大海
  - 脆い空洞
  - 木の炉辺
